# Security Policy

## Project Status

This project is in **alpha**. There are no API stability or security guarantees at this stage.

## Reporting a Vulnerability

**Do not open public issues for security vulnerabilities.**

Please use [GitHub Private Vulnerability Reporting](https://github.com/kaasops/dataprepper-operator/security/advisories/new) (Security tab > Report a vulnerability).

Response time: best effort. We aim to acknowledge reports within 7 days.
